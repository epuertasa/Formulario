**FixItPro** és una plataforma web corporativa d'elit dissenyada per a serveis de manteniment i suport informàtic. El projecte se centra a oferir una experiència d'usuari premium, neta i eficient, ideal per a empreses de tecnologia i professionals de la microinformàtica.

---

## 🚀 Característiques Principals

### 📋 Sistema de Gestió d'Incidències
Un formulari tècnic avançat (`suport.html`) que permet als clients reportar avaries de manera detallada:
- Identificació de l'equip i sistema operatiu.
- Selecció de components afectats (Hardware, RAM, Disc, etc.).
- Descripció detallada i nivell d'urgència.

### 💎 Disseny d'Alta Qualitat
- **Estètica Corporate**: Paleta de colors blaus i blancs que transmeten confiança i professionalitat.
- **Responsive Design**: Totalment adaptat a dispositius mòbils, tablets i escriptoris.
- **Tipografia Moderna**: Ús de les fonts *Inter* i *Outfit* per a una llegibilitat màxima.

### 🛠️ Seccions Incloses
- **Pàgina d'Inici**: Hero landing amb crides a l'acció (CTA).
- **Tarifes i Serveis**: Desglossament transparent de preus i plans SLA per a empreses.
- **Qui Som**: Secció corporativa detallant l'experiència de l'equip.
- **Contacte**: Formulari de contacte ràpid i integració de mapa de Google Maps.

## 📂 Estructura del Projecte

El codi està organitzat seguint les millors pràctiques de desenvolupament:

```bash
WB FORMULARIO/
├── css/
│   └── style.css      # Estils globals i components premium
├── js/
│   └── script.js      # Lògica d'interacció i validacions
├── img/
│   ├── header.png     # Identitat corporativa
│   └── erik.png       # Imatges de perfil/equip
├── index.html         # Portal principal
├── suport.html        # Formulari de tiquets
├── tarifes.html       # Serveis i preus
├── quisom.html        # Sobre l'empresa
└── contacte.html      # Canal de comunicació
```

---

## 🛠️ Tecnologies Utilitzades

- **Frontend**: HTML5 Semàntic, Vanilla CSS3 (Custom Properties).
- **Iconografia**: FontAwesome 6.4.0.
- **Fonts**: Google Fonts (Inter & Outfit).
- **Interaccions**: Vanilla JavaScript.
